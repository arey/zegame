1. Demo

System Requirements

The recommanded configuration to run the demo is :
(see game documentation for more details)

  • CPU : Pentium II  350Mhz
  • RAM : 96Mb de RAM
  • OS : PC Windows™ 95/98/2000
  • HD : 50Mb free space to install it
  • Video : 2Mb Video card 2D accelerated
  • Sound : Sound card compatible DirectSound (optionnal)
  • Joypad (optionnal)
  • DirectX 6.1 or above

Test Platforms

The demo has been tested on 3 different configurations :

  • PII 350Mhz 196Mb - Matrox G200  8Mb - Sound Blaster 64
           => development platform, recommanded required configuration (avg 40 fps)
  • P166 Mhz 48Mb - Matrox Millenium 1  2Mb - Ultra Sound
           => minimum configuration, small resolution (640x480), barely playable (avg 14 fps)
  • Laptop PIII 650Mhz 128Mb - NeoMagic  6Mb - Yamaha DS-XG
         

History

Here are some information about the public released that you could have downloaded in the past on this site. You can again download previous version that I let to your hand if you are interested.

1st release - alpha 1.00 - 12/30/1999

In this first release you could already admire the real first level of ZeGame (big.niv). You can go for a walk with the arrow keys or with a good old-style joypad, as you want. You can try to put1 or 2 swordthrusts, shot a burst with an old shotgun or kill some Ogre (if you can :-) 'cause in reality, they are just here as simple decoration.

Apart some scripts that animate the pigs and the locomotive, you could try to break some vases located near the start point (use the space key) : it will shatter and then a dwarf will exit and weak up.

 
2nd release - alpha 1.62 - 04/19/2000

A new level (new.niv) has been specially designed for this release where you will discover a large set of new features in comparison with version 1.00a. Here are few characteristics :

    • new animations for the Hero : run, jump, sword anims, ...
    • other characters : dwarf, robot, monk, ...
    • new interactions with the environment : kill ogres, break robot, activate traps, ...
    • 2 original musics with better quality,
    • ... and many other things you will discover by yourself ...

For information, the turning demo that I exposed at Milia had the 1.51a version number. The level was 3 times less big, there was less things to do and there were a lot of bugs. That's why I prefered wait to improve it a little bit before its release. On the other hand, there were not 2 musics but 12 musics from Guy-Roger as ambience music, to hold back the tourists for a full hour.

3rd release - alpha 1.70 - 10/05/2000

Released at the same time as the new update of the web site, this version does not include a lot of new features since version 1.61a. The main improvement is its size. Now the compressed file takes less than 12Mo thanks to the mp3 format. This is important for the large amount of people that continues to use a modem and wants to download ZeGame.

    • 2 previous music now at mp3 format with best audio quality
    • new sounds for musical ambiance & sound effects
    • joystick analogic almost supported
    • force feedback

Last release - alpha 1.75 - 06/05/2001

This version you'll find a way to play ZeGame with other players via the Internet. If you're connected to the Net it will automatically detect if a server is available somewhere on the earth and join this session. Else your computer will become a host for all other futur players.

2. Source Code

Game sources

Some people have already asked me to help them either with DirectX programming or with 3D isometric engine. So I already shared some of the source of ZeGame. And now I place all those sources at every body's disposal.

I've also created an account on SourceForge last june. It's an open source association that offer you a place on their debian linux server for your project. So I have right now an access to a cvs repository trough ssh on http://zegame.sourceforge.net where I will also copy ZeGame's source files.

At the beginning ZeGame was developing on C++Builder 1 from Borland. But during the summer 1999 I migrate to Visual C++ 6, even if the Tools remaines on the first environment. So, with the sources, I've also include the visual project files.
