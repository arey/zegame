ZeGame Documentation

 

In the following pages you will find a set of documentation pages related to ZeGame, according to what you want to do :

  • User game manual for all those who wants to know how to play to ZeGame,
  • Tools documentation that explains how to use a set of tools specially designed for Nains
    • Level design doc : if you want to build a game level by yourself,
    • Tile editor doc : that explains you how import your own graphics,
    • Animation editor doc : if you want to create new characters or objects,
  • Files specifications give you a better idea of the role of each file and how modify them,
  • Architecture and Conception overview of ZeGame
