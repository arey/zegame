User Guide

1. Introduction

ZeGame will immerse you in a full gaming experience in an epic fantasy virtual world. This is a single role playing game with 3D isometric view. And the story takes place a long time ago in a medieval world where some inhabitant have the secret of strange kind of steam engines of our late XIXth century.

2. Game Features

  • Powerfull graphic engine in 3D isometric : transluency, shadows, infinite levels, ...
  • A large set of character animations : walk, run, jump, hit, ...
  • Fairy musical athmosphere and immersive sound effects
  • 2 full levels right now available

4. System Requirements

Minimum system configuration required to run ZeGame are as follows ...

  • Intel Pentium® 200 or 100% compatible CPU (PII 300 recommended)
  • 64 Mb RAM (96 Mb recommended)
  • 2 MbVideo card accelerated 2D (8Mb is better)
  • 50 MB available hard drive spac
  • Microsoft Windows® 95/98 or 2000
  • DirectX™ 6.1 or higher

Optionnal specs :

  • Sound card compatible with DirectSound
  • 4 buttons Joypad
