Map Editor

This tool is a powerful graphical tool created specially for ZeGame. It is useful to design the different level maps of this game like big, milia or new that you can find in the current version of Nains.

New level loaded in the MapEditor

It can be used by anybody and allows to build easily a map by selecting tiles and place them on the same isometric 3D view you have during the game. So if you are an infographist, a hard-core gamer, a developper and (of course) a game designer, you can use this tool to create your own levels and integrate them by yourself into ZeGame.

1. User Guide

1) General presentation of the interface

The editor is divided in main areas :

  • the map view, located at the left top corner of the editor, take 3/4 of the screen. It is a 3D isometric representation of a level, the same view that you will have in the game. And it is on this view that you could place the tiles and the characters you want to build a level.

  • the map tools, situated at the bottom of the screen, is divided in 2 sections : the left one give you information about the map and the right one allow you to do several actions as select an object on the map view.

  • the lib utils, located at he right top corner of the screen, allow you to choose with what type of component you want to work : characters, objects, background, ...

  • the object palette, placed at the right side of the editor, give you the oppurtunity to work accurately with a kind of object.

The menu give you access to a lot of functions to create, load or save a level as well as create your level tile library.

2) How to start ?

This editor allow you to create level maps for ZeGame. This chapter will try to explain you how to process.

A) Map creation

To create a new map go in the menu Fichiers\Nouvelle Carte. The following dialog box will appear :



New map creation dialog box

Here you have to give several informations :
- the name of your map (Nom de la Carte),
- a ID number for this map (Numero de la Carte),
- as well as its dimensions in number of trapezoid (Dimension de la Carte).

After you click on the Creer button, an empty wire grid will appear.

Before doing anything, I advise you to select a tile library in the Lib Outils panel. See the Create Level Lib section for more information about how to create a special tile library for your own map. A file chooser dialog box must appear and you can select the .LiL file you want.

B) Tile Level Library
Preamble

When I started the game I did a lot of difficult things that I would have never done. Create a tile library specifically for a level is part of those things. It should have helped to only load the tiles required by this level with the aim of optimize the game, but I could do it automatically instead of bothering the level designer … Fortunately it provides of way to select a set of tiles among a lot.

Presentation

From the menu you can open, load or create a level tile library. By default the dialog opens in new level tile library mode.
Like you can see the panel is divided in 2 lists. The left list is for your level library. And the right list shows all the tiles you have in the universal tile library.

Create a new library

The first thing you should do is to enter the name of your library in the Librarie du Niveau field. Then you can select the tiles you want use in your map from the right list : you can select a tile and click on the Ajouter button or click on the Ajouter tout button to add all tiles from the right list in your map library.

 


Level tile level creation dialog box

The tiles of the universal library are classified in 2 different kinds :
- tiles used to draw the background (Decors) : tables, trees, walls, objects, …
- and tiles used to draw the ground (Sols) : bade of grass, soil, paving stones …


In both of those 2 kinds of tile, you have several sections for :
- natural tiles (Natur) : trees, bushes, grass, …
- object tiles (Objet) : table, robot, wheel, …
- wall tiles (Mur) : barrier, gate, rocks, …
- architectural tiles (Archi) : house, vase, hut, …

Those tiles are again divided in 2 tabs :
- underground cave (Grotte)
- plain (Plaine)

When you save your library, it generates a .LiL file that has the name you gave and that is stored in the Level\ directory.

C) Universal Tile Library
After you add a tile (bitmap file and .dec file) in its directory, you have to edit the Universal Tile Library (go in the Lib\ menu) and add your new tile in the category you want. You can also add in Tiles\IDTiles.id its ID if you didn't do that.

The universal tile library is stored in the .LiD files in the Tiles\ directory. There are 14 files, 7 for the plain tiles and 7 for the cave tiles. In the 7 files, you have 4 files for the object of the background (…dArchi, …dNatur, …Objet, …Mur) and 3 files for the ground tiles (…sArchi, …sNatur, …Inter). So load the file corresponding to the category you want, add your tiles and save it.

Incoming features
For the moment, the categories are hard-coded. So you cannot add a new one without going in the C++ files (Consts.h more exactly). In the future I like to pass those constants in resources files you can change as you want.

3) Designing a level

Once you have created a new map or just load an existing one, you can build it as you want : draw the entire map, add characters or ground zones, assign a script to an object, test the pathfinder and so more.

The map editor screen is made up of 4 panels :

a) MapView panel where you can see the map like it will be in the game,
b) MapTools panel give you some information about the current map and allow you to place, select or delete objects on the map,
c) LibTools panel allow you to select on what kind of things you want to work : on characters, background, ground zones …
d) ObjectTools is a panel that indeed depends on what work space you are working.

A) How to draw the Background Floor ?

Once you select a tile library for the level, as above-mentioned in the MapCreation section, the first thing you could do is to draw a part of the floor of your new empty level.

So you have to click on the Fond button at the top of the LibTools panel. Then you must select one of the three types of floor you want to use (sNatur, sArchi or Inter). Left click on the Placer button on the Actions panel in the MapTools and position your mouse on the MapView where you want to place this floor tile. Just one more step : left clic on the view. You just placed a floor tile.
You can continue with this same type of floor or select another one with the aim to continue to draw the background of your level.

An advanced function allow you to paint the ground with the same tile easily. Use the Peindre button instead of the Placer button. And when you want to draw the ground let the left button pushed and move your mouse where you want.

B) How to place an Object on the map ?

If you read the previous section, you know how to place a background tile so it will be very easy to put an object tile on the map.
Click on the Decors button at the top of the LibTools panel. Then choose one of the four kind of object (dNatur, Objet, Mur or dArchi). Use the Placer button in the MapTools panel and place your selected object where you want on the map by a simple left-click.

Note that you cannot stack 2 object tiles (or 2 floor tiles) one above the other. In that case the trapezoids of your tile will become read instead of yellow.

C) How to remove something from the map ?

To delete any tile you have previously put on the map, just click on the Select button in the MapTools panel, then select the tile you want to remove and click on the Suppr button of the MappTools panel.

Tips
Note that you can use the contextual menu by right-clicking once you have selected the object you want to remove. And choose the Supprimer objet menu.
