Files
~~~~~
 - Devkit.txt	    :  this text file
 - MapEditor.exe    :  game level editor
 - AnimEditor.exe   :  character & object animation editor
 - CreateDecors.exe :  bitmaps import tool
 - BuildPakFile.exe :  ZGP files builder
 - Pak*.ini	    :  samples of ini files used by BuildPakFile.exe
 - GameDoc\	    :  html doc that explains how using those tools

Install
~~~~~~~

You have to unrar those files in the folder where are installed all the
resources (bitmap, level, sound, data ...) of ZeGame. You can dowmload
those resources file from the follwowing url : 
http://www.zegame.net/download/ZeGame-Res-v1.70.rar


Use
~~~

See Gamedoc\gamedoc.html or web site that refer how use all those tools.
