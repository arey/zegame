Description
~~~~~~~~~~~
You can find here all the resources files used for ZeGame : graphics,
sound effect, music, anims ...


Install
~~~~~~~

If you want to use either any tools you can find in the Development Kit or 
to compile the sources without using .zgp files, you have to unrar this file
in the ZeGame\ sub-directory where you installed the sources.
